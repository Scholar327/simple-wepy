# 小程序 -- 基于 wepy 框架

## 开发前准备

<p style="color: red">开发前请先认真阅读<a href="./CONTRIBUTING.md">开发指南</a></p>

``` bash
# 项目使用minui做组件库，故需先全局安装min-cli
npm install -g @mindev/min-cli

# 安装模块依赖
npm install

# 编译 min-ui 组件
min build
```

## Build Setup

``` bash
# watch file change
npm run dev

# build for production with minification
npm run build
```
